# HungerVerse AI 🌍🍱

HungerVerse AI is an innovative smart web platform designed to **predict hunger-prone areas, reduce food waste, and connect surplus food donors with NGOs** using Artificial Intelligence and real-time analytics.

This project was developed for **Hackathon / Open Innovation Challenge**.

---

## 🚀 Features

### 🧠 AI Hunger Prediction Dashboard
- Predicts hunger risk levels in cities
- Displays High / Medium / Low risk zones
- Helps NGOs act early

### 🍱 Donate Food Portal
- Users can donate extra food
- Submit food name, quantity, location, expiry time

### 🗺️ Google Maps Integration
- Shows nearby food donation locations
- Live map markers for cities

### 📊 Live Hunger Analytics
- Charts and analytics dashboard
- Smart visualization using Chart.js

### 🚨 Emergency NGO Alert
- Sends instant alerts when hunger risk is high

### 🔐 Login Portal
- User Login
- NGO Login
